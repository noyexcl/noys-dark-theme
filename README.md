# Noys Dark Theme

A premium, rich dark theme for Visual Studio Code featuring vibrant accents, clean syntax highlighting, and a sleek modern aesthetic.

![Noys Dark Theme Showcase](https://raw.githubusercontent.com/noyexcl/noys-dark-theme/main/images/showcase.png)

## Design Philosophy

* **Rich UI Backgrounds**: Replaces generic pure blacks or grays with curated deep charcoal and slate tones to reduce eye strain while maintaining a high-contrast layout.
* **Modern Accents**: Employs a stunning combination of deep violet/indigo and vibrant teal for focus indicators, selection states, and primary actions.
* **Balanced Syntax**: A carefully balanced palette of pastel-neon colors (soft pinks, warm oranges, minty greens, and sky blue) makes reading code for hours effortless.

## Features

- Rich HSL-tailored dark color scheme.
- High accessibility and readability.
- Support for multiple languages (JavaScript, TypeScript, Python, HTML/CSS, Rust, Go, and more).

## Installation

1. Open **Visual Studio Code**.
2. Go to **Extensions** (`Ctrl+Shift+X` or `Cmd+Shift+X`).
3. Search for `Noys Dark Theme`.
4. Click **Install**.
5. Select `Noys Dark Theme` when prompted for the color theme.

## Customization

If you want to customize specific colors, you can override them in your `settings.json`:

```json
"workbench.colorCustomizations": {
    "[Noys Dark Theme]": {
        "editor.background": "#141416"
    }
}
```
